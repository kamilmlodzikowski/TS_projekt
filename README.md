# Wredny paczkomat
Projekt z Teorii Sterowania

Robot obsługuje paczkomat, w którym można nadawać i odbierać paczki.

Zespół w składzie: Maja Mrozińska, Marcel Koczorowski, Kamil Młodzikowski
